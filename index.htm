Tutorial for another way of setting up the formatting

Right Click is Key

Right click email. Select Rules > Create Rule.

2

Move to Folder

Move emails from this sender to a specific folder (e.g., "Newsletters").

3

Color Code the Boss

In Conditional Formatting, change specific people to RED.

WARNING!

This way of setting up rules works only on desktop and cannot be easily done on the phone website or mobile application.
